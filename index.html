Теги

< meta > - зберігає інформацію призначену для браузерів і пошукових систем.

< h1 - h6 > - Теги від < h1 > до < h6 > використовуються для визначення заголовків HTML. < h1 > визначає найважливіший заголовок. < h6 > визначає найменш важливий заголовок.

< pre > - Тег <pre> визначає попередньо відформатований текст.

< p > - Тег <p> визначає абзац. Браузери автоматично додають один порожній рядок перед і після кожного елемента <p>.

< html > - Тег <html> представляє корінь документа HTML.

< head > - Елемент <head> є контейнером для метаданих (даних про дані) і розміщується між тегами <html> і тегами <body>.

< title > - Тег <title> визначає назву документа.

< link > - Тег <link> визначає зв'язок між поточним документом і зовнішнім ресурсом.

< header > - Елемент <header> представляє собою контейнер для вступного вмісту або набір навігаційних посилань.

< body > - Тег <body> визначає тіло документа.

< footer > - Тег <footer> визначає нижній колонтитул для документа або розділу.
