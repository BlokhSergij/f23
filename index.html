Довідник по HTML тегам

<!DOCTYPE> - визначає тип документа та надає основну інформацію для браузера — мову документу та його версію.

<html> - корeневий елемент HTML документа. Повідомляє браузер, що це HTML-документ. Являється контейнером для всіх інших html-елементів.

<head> - елемент-контейнер для метаданих HTML-документів, таких як <title>, <meta>, <script>, <link>, <style>.

<meta> - служить для зберігання додаткової інформації про сторінку. Цю інформацію використовують браузери для обробки сторінок, а пошукові системи для її індексації.

<title> - заголовок HTML - документа, який відображається у верхній частині рядка заголовка браузера.

<link> - визначає зв'язок між документом та зовнішнім ресурсом. Також використовується для підключення зовнішніх таблиць стилів.

<style> - підключає вбудовувані таблиці стилів.

<header> - розділ для ввідної інформації сайту або групи навігаційних посилань. Може містити один або декілька заголовків, інформацію про автора.

<body> - являє собою тіло документа.

<h1-h6> - являють собою 6 рівнів заголовків, де <h1> це найбільший заголовок, а <h6>-найменший.

<p> - визначає текстовий абзац.

<b> - робить текст жирним.

<footer> - визначає завершальну область (нижній колонтитул) документа чи розділу.
