Справочник по HTML тегам

<!DOCTYPE> - предназначен для указания типа текущего документа;

<html> - контейнер, который заключает в себе все содержимое веб-страницы;

<head> - содержит машиночитаемую информацию (metadata) о документе, например его заголовок, скрипты и страницы стилей;

<meta> - позволяет сообщить браузеру посетителя, какой набор символов и какую кодировку необходимо установить на веб-странице;
Атрибут charset - задает кодировку символов для HTML документа

<title> - определяет заголовок документа, который отображается в заголовке окна браузера или на вкладке страницы;

<link> - определяет отношения между текущим документом и внешним ресурсом;
Атрибут href - путь к связываемому файлу;
Атрибут rel - onределяет отношения между текущим документом и файлом, на который делается ссылка;

<style> - содержит стилевую информацию для документа или его части;

<header> - задает «шапку» сайта или раздела, в которой обычно располагается заголовок;

<body> - предназначен для хранения содержания веб-страницы (контента), отображаемого в окне браузера;

<h1> - это HTML-заголовок, который чаще всего используется для разметки заголовка веб-страницы, h2, h3, h4, h5, h6 и такие же заголовки только меньше;

<p> - представляет собой абзац;

<pre> - определяет блок предварительно форматированного текста;

<footer> - представляет собой нижний колонтитул (футер, подвал);
